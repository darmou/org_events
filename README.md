# README For Org Events Project

To get this application up and running.

* Ruby version
ruby 2.3.0p0 (2015-12-25 revision 53290) [x86_64-darwin15]

* Install Dependancies
```
bundle install
```

* Database creation
```
rake db:create
rake db:migrate
```

* How to run the test suite
```
rails test
```

* Starting the application
```
rails s
```
